# 台灣會計軟體功能拆解與 OCR 報表辨識規格

日期：2026-05-22

## 研究摘要

台灣中小企業與記帳士使用的會計軟體，通常不是單純「記分錄」，而是把憑證蒐集、電子發票、進銷存、總帳、帳款、營業稅、營所稅、月結報表與文件保存串成一條流程。市場上的功能樣本可分成三類：

1. 傳統 ERP/財會系統：重點在進銷存、帳款、總帳、財報與多部門/多幣別管理，例如鼎新 A1、正航、文中等。
2. 稅務申報系統：重點在電子發票匯入、進銷項整理、401/403 申報、BAN.TXT/TET_U 等媒體檔，例如報稅通、文中發票管理。
3. 會計師/記帳士雲端工作台：重點在 OCR、客戶上傳、批次建帳、自動對帳、月結報表、歸檔與簽核，例如 CPA Cloud 類型產品。

本產品應採「會計核心 + 稅務合規 + OCR 文件入口」架構，先做可驗證的帳務與報表資料，再逐步擴充進銷存、電子發票 API、營所稅與事務所協作。

## 核心功能地圖

### 1. 基礎資料與權限

- 公司資料：統一編號、稅籍編號、公司名稱、負責人、營業地址、行業別、稅務申報類型。
- 多公司/多分支：支援同一帳號管理多家公司、總分支機構、彙總報繳。
- 使用者角色：老闆、會計、記帳士、審核者、唯讀客戶。
- 會計科目：台灣常用科目表、科目代碼、部門/專案/成本中心。
- 期間管理：月結、年結、申報後鎖定、反結帳流程與異動紀錄。
- 匯入匯出：Excel/CSV 匯入客戶、廠商、商品、期初餘額、發票與傳票。

### 2. 憑證與文件入口

- 支援檔案：圖片、PDF、掃描檔、Excel、CSV。
- 文件類型：統一發票、電子發票、收據、合約、銀行明細、401/403/404、資產負債表、損益表、現金流量表、固定資產表、庫存表。
- OCR 建帳：自動擷取統編、日期、發票號碼、金額、稅額、供應商/客戶、品項、報表欄位。
- 低信心度複核：欄位信心度不足時標示待確認，修改後回寫模型規則。
- 文件歸檔：依公司、年度、月份、文件類型、申報期別分類；支援全文搜尋。

### 3. 總帳與傳票

- 傳票管理：收入、支出、轉帳、調整、折舊、結轉傳票。
- 自動分錄：由發票、銀行交易、薪資、折舊、進銷存單據自動產生借貸分錄。
- 帳簿：日記帳、明細分類帳、總分類帳、試算表。
- 勾稽：借貸平衡、科目餘額方向、期初期末連續性、負數/異常科目提醒。
- 結帳：月結、年結、損益結轉、本期損益轉保留盈餘。

### 4. 應收應付與資金

- 客戶/廠商主檔：統編、付款條件、信用額度、聯絡資訊。
- 應收帳款：請款、收款、沖帳、逾期提醒、帳齡分析。
- 應付帳款：付款排程、批次付款、沖帳、付款憑證。
- 銀行與現金：銀行帳戶、現金帳、零用金、銀行交易 CSV 匯入。
- 自動對帳：以金額、日期、摘要、對象比對銀行明細與帳務紀錄。

### 5. 進銷存與訂單

- 商品主檔：品號、品名、規格、單位、售價、成本、稅別。
- 銷售流程：報價、訂單、出貨、銷貨、退貨、折讓、收款。
- 採購流程：請購、採購、進貨、退貨、折讓、付款。
- 庫存：入出庫、調撥、盤點、期末庫存、成本計算。
- 報表：訂單未出明細、商品銷售統計、庫存明細、毛利分析。

### 6. 發票與電子發票

- 進項發票：手動輸入、OCR、CSV、電子發票平台匯入。
- 銷項發票：字軌/號碼管理、開立、作廢、折讓、註銷。
- 電子發票串接：財政部電子發票整合服務平台、Turnkey 或加值中心 API。
- 重複檢查：發票號碼、統編、期別、金額比對。
- 扣抵判斷：營業稅法第 19 條不可扣抵項目標記，例如交際費、員工福利、自用車相關費用。
- 發票與傳票檢核：進項稅額、銷項稅額與會計科目互相勾稽。

### 7. 營業稅與 401/403/404

- 申報類型：401 一般稅額計算、403 兼營或特種、404 專營特種。
- 申報頻率：一般按期申報；核准者可按月申報。
- 401 核心欄位：
  - 所屬年月份、統一編號、稅籍編號、營業人名稱。
  - 應稅銷售額、零稅率銷售額、銷項稅額。
  - 進貨及費用、固定資產、進項稅額。
  - 退出折讓、海關退還溢繳稅款、不得扣抵進項。
  - 本期應納稅額、留抵稅額、應退稅額。
  - 附件份數、申報日期、代理申報資訊。
- 媒體檔：支援 BAN.TXT、TET_U 等申報檔產生與格式檢核。
- 報表：統一發票明細表、進銷項明細、稅額試算、401/403 主表、繳款書資料。

### 8. 固定資產

- 資產主檔：資產編號、類別、取得日期、取得成本、耐用年限、殘值、保管部門。
- 折舊方法：直線法、定率遞減法。
- 異動：增添、出售、報廢、減損、重分類。
- 報表：財產目錄、折舊明細、資產餘額、處分損益。
- 稅務連動：營所稅財產目錄、折舊限額、固定資產進項稅額。

### 9. 薪資、扣繳與二代健保

- 員工/所得人主檔：身分證字號、居住者身分、所得類別、投保資料。
- 給付作業：薪資、租金、執行業務所得、股利、其他所得。
- 扣繳計算：扣繳稅額、二代健保補充保費、免扣繳判斷。
- 年度申報：扣繳憑單、媒體申報 CSV、給付總表。

### 10. 營所稅與年度申報

- 年度損益：由總帳轉入營利事業所得稅申報基礎資料。
- 申報方式比較：擴大書審、查帳申報、會計師簽證。
- 年度附表：資產負債表、損益及稅額計算表、營業成本明細、財產目錄、未分配盈餘。
- 調整項：交際費限額、折舊限額、免稅收入、投資收益、暫時性差異。
- 繳款書與附件：條碼繳款書資料、附件掃描與歸檔。

### 11. 月度與年度報表中心

月度報表：

- 試算表。
- 資產負債表。
- 綜合損益表。
- 現金流量表。
- 應收帳款帳齡表。
- 應付帳款帳齡表。
- 銀行調節表。
- 進銷項發票明細。
- 營業稅試算表與 401 草稿。
- 費用明細表。
- 固定資產折舊月報。
- 庫存明細與毛利表。
- 客戶/部門/專案損益。

年度報表：

- 兩年度對照資產負債表。
- 兩年度對照損益表。
- 現金流量表。
- 權益變動表或盈虧撥補表。
- 財務報表附註。
- 年度試算表。
- 財產目錄。
- 營業成本明細表。
- 未分配盈餘申報資料。
- 扣繳憑單彙總。
- 營所稅申報書與附表。

### 12. 審核、簽核與稽核軌跡

- 審核流程：上傳、辨識、建議分錄、會計複核、主管簽核、入帳、鎖定。
- 異常規則：重複發票、統編錯誤、稅額不等於銷售額 5%、報表不平、期別不一致。
- 稽核軌跡：建立者、修改者、修改前後值、核准者、時間戳記。
- 版本控管：每次申報檔、報表、OCR 結果與人工修正皆保存版本。

## OCR 報表辨識設計

### OCR 流程

1. 上傳檔案：圖片、PDF 或掃描資料。
2. 影像前處理：旋轉校正、去噪、裁切、表格線偵測、頁面分割。
3. 文件分類：判斷是 401 表、資產負債表、損益表、銀行明細、發票、固定資產表等。
4. 文字與表格辨識：OCR 文字、表格格線、欄列關係、數字格式。
5. 欄位抽取：依模板、關鍵字、座標、表格標題與語意模型抽欄位。
6. 標準化：民國年轉西元年、千分位處理、負數格式、科目名稱對應、統編格式。
7. 勾稽驗證：報表平衡、欄位合計、稅額試算、跨報表比對。
8. 人工複核：低信心度欄位、勾稽失敗欄位、未知科目需人工確認。
9. 入帳/歸檔：產生分錄、更新報表資料、保存原始檔與辨識結果。

### OCR 文件類型與欄位

#### A. 資產負債表

必抓欄位：

- 公司名稱、統一編號、報表日期、幣別、金額單位。
- 資產：流動資產、現金及約當現金、應收帳款、存貨、預付款、固定資產、累計折舊、其他資產、資產總額。
- 負債：流動負債、應付帳款、應付稅款、短期借款、長期負債、負債總額。
- 權益：股本、資本公積、保留盈餘、本期損益、權益總額。
- 比較期：本期、上期、去年同期。

驗證規則：

- 資產總額 = 負債總額 + 權益總額。
- 明細合計 = 分類小計。
- 本期期初應銜接前期期末。
- 資產負債表本期損益應可對應損益表稅後淨利。

#### B. 401 營業人銷售額與稅額申報書

必抓欄位：

- 報表代號、所屬年月份、按月/按期、總機構彙總註記。
- 統一編號、稅籍編號、營業人名稱、負責人、地址。
- 應稅銷售額、零稅率銷售額、銷項稅額。
- 進貨及費用金額、固定資產金額、進項稅額。
- 退出折讓、海關退還溢繳稅款。
- 不得扣抵進項稅額、可扣抵進項稅額。
- 本期應實繳稅額、留抵稅額、申請退稅額。
- 附件份數、申報日期、申報單位、代理人資訊。

驗證規則：

- 銷項稅額應接近應稅銷售額 x 5%，需容忍四捨五入與特殊欄位。
- 可扣抵進項不得包含第 19 條不可扣抵項目。
- 401 銷售額應與銷項發票、損益表營業收入互相勾稽。
- 進項稅額應與進項發票、費用/資產科目互相勾稽。
- 報表期別不可與已鎖定申報資料衝突。

#### C. 損益表 / 綜合損益表

必抓欄位：

- 營業收入、銷貨退回及折讓、營業收入淨額。
- 銷貨成本、營業毛利。
- 營業費用：薪資、租金、文具、旅費、交際費、折舊、廣告、其他費用。
- 營業利益、營業外收入、營業外費用。
- 稅前淨利、所得稅費用、稅後淨利。

驗證規則：

- 收入 - 成本 - 費用 +/- 營業外項目 = 稅前淨利。
- 稅後淨利應可銜接資產負債表權益變動。
- 營業收入應與銷項發票/401 銷售額建立差異表。

#### D. 現金流量表

必抓欄位：

- 營業活動現金流量。
- 投資活動現金流量。
- 籌資活動現金流量。
- 本期現金增減數。
- 期初現金及約當現金。
- 期末現金及約當現金。

驗證規則：

- 期初現金 + 本期現金增減數 = 期末現金。
- 期末現金應與資產負債表現金及約當現金一致。

#### E. 月度雜項報表

優先支援：

- 銀行對帳單：帳號、交易日、摘要、支出、收入、餘額。
- 應收帳款表：客戶、發票/單據、到期日、未收金額、逾期天數。
- 應付帳款表：廠商、發票/單據、到期日、未付金額、逾期天數。
- 庫存表：品號、品名、期初、進貨、銷貨、調整、期末、單位成本、庫存金額。
- 固定資產折舊表：資產編號、取得成本、累計折舊、本期折舊、帳面價值。
- 費用明細表：日期、摘要、科目、金額、稅額、憑證號碼。

#### F. 年度雜項報表

優先支援：

- 營所稅申報書頁面：損益及稅額計算表、資產負債表、營業成本明細、財產目錄、盈餘分配/盈虧撥補。
- 扣繳憑單彙總：所得人、所得類別、給付總額、扣繳稅額、二代健保。
- 財報附註：重要會計政策、關係人交易、重大承諾與或有事項。
- 年度調整表：帳載金額、稅務調整、申報金額。

## 建議 MVP 範圍

### 第一階段：OCR 報表資料庫

- 建立公司主檔、期間、文件上傳、OCR 任務、人工複核畫面。
- 支援資產負債表與 401 表 OCR。
- 做欄位抽取、信心度、錯誤標示、原圖對照。
- 建立資產負債表與 401 的驗證規則。
- 產生月度報表資料卡與 PDF/Excel 匯出。

### 第二階段：憑證到營業稅

- 加入統一發票/電子發票 OCR 與 CSV 匯入。
- 建立進銷項發票管理、重複檢查、扣抵判斷。
- 由進銷項資料產生 401 草稿與 BAN.TXT/TET_U。
- 加入申報後期間鎖定與版本保存。

### 第三階段：完整會計循環

- 加入傳票、總帳、試算表、損益表。
- 加入應收應付、銀行對帳、固定資產折舊。
- 報表中心支援月結包與年度申報包。

### 第四階段：事務所/多客戶協作

- 客戶上傳入口。
- 記帳士工作佇列。
- 批次審核與主管簽核。
- 多公司合併、權限隔離、稽核軌跡。
